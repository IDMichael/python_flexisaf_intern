# Smart Student Grade Calculator

What the Program Does:
This program is a command-line Python application that calculates a student’s academic performance.

It:

* Prompts the user to enter test, assignment, and exam scores 
* Validates that each score is a number between 0 and 100
* Calculates the total and average score
* Determines whether the student passed or failed
* Assigns an award status (Excellence, Merit, or No Award) based on the average and exam score
* Allows the user to repeat the calculation or exit the program
* Allows the user to cancel input at any stage using quit or q

# How to Run the Program
1. Ensure Python 3 is installed:
python --version

2. Save the file (e.g. grade_calculator.py)

3. Run the program:

python grade_calculator.py
or:
python3 grade_calculator.py

Follow the on-screen prompts to enter scores.

# Required Libraries
* None

The program uses only Python’s built-in functions and does not require any external libraries.
